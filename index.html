pysb

Python framework for Systems Biology modeling
PySB is a framework for building rule-based mathematical models of biochemical systems. It works nicely with scientific Python libraries such as NumPy, SciPy and SymPy for model simulation and analysis.

Documentation

All of the PySB documentation is available online, courtesy of Read the Docs.

Download

You can get the latest release of PySB from the Python Package Index. However there are quite a few dependencies and not all of them are other Python packages, so it can be a bit tricky to install.

We also provide a live DVD running the Ubuntu distribution of Linux with PySB pre-installed. This will let you try PySB without installing its many dependencies yourself. Download the ISO image and burn it onto a DVD, then reboot your computer with the DVD still in the drive. Beware that any files you create will be lost on reboot, so use a USB key to save your work. Or you can boot the ISO file directly in VirtualBox without requiring a physical DVD.

You can clone the source code with Git by running: 

$ git clone git://github.com/pysb/pysb
